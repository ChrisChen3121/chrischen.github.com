* 层叠次序
1) 浏览器缺省设置
2) 外部样式表
3) 内部样式表（位于 <head> 标签内部）
4) 内联样式（在 HTML 元素内部）
* 选择器
** 派生选择器
#+begin_src css
  h2 strong {
      color: blue;
  }
#+end_src
#+begin_src html
  <h2>The strongly emphasized word in this subhead is<strong>blue</strong>.</h2>
#+end_src
** id选择器
#+begin_src css
#red {color:red;}
#green {color:green;}
#+end_src
#+begin_src html
<p id="red">这个段落是红色。</p>
<p id="green">这个段落是绿色。</p>
#+end_src
id属性每个html只能出现一次
#+begin_src css
  #sidebar p {
      font-style: italic;
      text-align: right;
      margin-top: 0.5em;
  }
#+end_src
** 类选择器
#+begin_src css
  .center {text-align: center}
#+end_src
#+begin_src html
<h1 class="center">
This heading will be center-aligned
</h1>
<p class="center">
This paragraph will also be center-aligned.
</p>
#+end_src

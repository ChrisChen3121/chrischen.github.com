* 超链接打开新窗口
target="_blank"
#+begin_src html
  <a href="http://www.w3school.com.cn/" target="_blank">Visit W3School!</a>
#+end_src
* 定义Name或id属性，用于内部链接
#+begin_src html
  <a name="tips">基本的注意事项 - 有用的提示</a>
  <a href="#tips">有用的提示</a>
  <a href="http://www.w3school.com.cn/html/html_links.asp#tips">有用的提示</a>
#+end_src
第三行表示跳转到该网页内部的tips元素
* 子文件夹后始终添加正斜杠
#+BEGIN_VERSE
href="http://www.w3school.com.cn/html" Bad!
href="http://www.w3school.com.cn/html/" Good!
#+END_VERSE
不添加的话，会产生两次请求。
* <img>元素
- 无对应结尾符，只包含属性的元素。
- alt属性添加可替换文本。
  #+begin_src html
  <img src="boat.gif" alt="Big Boat">
  #+end_src
* class还是id?
class 用于元素组（类似的元素，或者可以理解为某一类元素），而 id 用于标识单独的唯一的元素。

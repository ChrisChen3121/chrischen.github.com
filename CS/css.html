2 选择器

2.1 派生选择器

h2 strong {
    color: blue;
}

<h2>The strongly emphasized word in this subhead is<strong>blue</strong>.</h2>

2.2 id选择器

#red {color:red;}
#green {color:green;}

<p id="red">这个段落是红色。</p>
<p id="green">这个段落是绿色。</p>

id属性每个html只能出现一次 

#sidebar p {
    font-style: italic;
    text-align: right;
    margin-top: 0.5em;
}

2.3 类选择器

.center {text-align: center}

<h1 class="center">
This heading will be center-aligned
</h1>
<p class="center">
This paragraph will also be center-aligned.
</p>

2.4 属性选择器

[title]
{
color:red;
}
[title=W3School]
{
border:5px solid blue;
}
